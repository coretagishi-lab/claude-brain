# Mistakes — 繰り返しミス記録

> 一度だけのミスは書かない。**同じミスを2回以上した**、または**確実に繰り返しそうな**ミスだけ記録する。

---

## フォーマット

```
### [YYYY-MM-DD] ミスのタイトル
- **状況**: どんな時に起きたか
- **ミス内容**: 何をやらかしたか
- **正しい対処**: 次回はこうする
- **発生回数**: X回
```

---

## 記録ルール（v4.0更新）

- **同じフィードバックが2回以上来たもの**のみここに記録する
- 1回だけは `review-notes.md` に記録して終わり
- ここに来た時点で「対策を仕組み化」することが目的
